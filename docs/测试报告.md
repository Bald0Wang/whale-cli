## MCP 验收

- `mcp.example.json` 使用 `mcpServers` 格式，可直接启动 `examples/mcp_echo_server.py`。
- `Soul` 默认工具池会加载 MCP adapters，并在退出、清空会话和切换会话时调用 `Soul.close()` 释放 lifecycle。
- 实现支持 `stdio`、Streamable HTTP（`http`）和 SSE（`sse`），兼容常见配置中的 `type` 字段；真实回归使用本地 stdio echo server，外部高德 SSE endpoint 已做发现和调用验证。OAuth 交互回调尚未实现，未作为已验收功能。

## 品牌改名验收
