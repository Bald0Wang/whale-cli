# Whale CLI 教程问题汇总（历史归档）

这份文件曾用于记录早期项目阶段的文档/代码不一致问题。

当前项目已经改名为 **Whale CLI**，并补齐了可运行的教学版模块：

- agent 配置模板
- hooks
- subagent
- background tasks
- skills 分层发现
- plugin loader
- AGENTS.md 项目上下文

当前权威入口：

- 教程总览：`docs/新手入门/README.md`
- 写作模板：`docs/教程小节标准化写作模板.md`
- 架构说明：`docs/结构说明.md`
- 测试报告：`docs/测试报告.md`

旧问题清单不再作为当前实现依据。后续发现的新问题，请直接在测试报告或 issue 中记录。
